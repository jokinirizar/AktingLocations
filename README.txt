Sencilla aplicación basada en react que permite al usuario crear una lista de países y ciudades visitadas con sus respectivas coordenadas.

Basándose en esas coordenadas la aplicación perimitirá localizar dichas ciudades en un mapamundi, cambiando de color dependiendode si han sido visitadas o no.

Redux y Redux-toolkit fueron utilizados para gestionar la información y el estado de la aplicación de forma centralizada, React Router para manejar el uso de múltiples páginas y leaflet para la visualización del mapa y sus marcadores.


Con la última versión de node instalada en el PC es suficiente para usar la aplicación. Tras hacer pull al repositorio ejecutar npm install en la consola desde la raíz del proyecto para instalar las dependencias.
Después ejecutar npm start para lanzar la aplicación en el servidor local.
